Cosmos SDK is a framework for building application-specific blockchains in the Go programming language. The development paradigm is different from virtual-machine-based blockchains like Ethereum to avoid the limitations of the {{< glossary "smartcontract" "smart contract" >}} model. 

The {{< glossary "laconic-network" "Laconic Network" >}} is based on the Cosmos SDK.
